🧾 주문 관리

고객 주문을 확인하고 수락하세요. 수락한 주문은 고객이 취소할 수 없어요.

일시 주문자 구성 금액 상태 관리
아직 들어온 주문이 없어요

📧 주문 수신 설정

고객이 주문하기를 누르면 이 이메일 주소로 주문서가 발송돼요.

💡 실제 자동 발송 설정법: web3forms.com에 주문받을 이메일을 입력하면 무료 액세스 키가 메일로 바로 와요. 그 키를 위에 붙여넣으면 고객이 주문할 때 메일 앱 없이 자동으로 이메일이 발송됩니다 (월 250건 무료). 키가 없으면 고객의 메일 앱이 열리는 방식으로 동작해요.

☁️ 모든 기기 동기화

관리자 키를 등록하면 여기서 수정한 상품·계정·설정이 모든 컴퓨터·모든 방문자에게 반영돼요. 키는 이 브라우저에만 저장됩니다.

💡 키 만들기 (1회, 2분): github.com/settings/personal-access-tokens/new 접속 → Token name: snackfit → Repository access에서 Only select repositories → snackfit-data 선택 → Permissions에서 Contents → Read and write → Generate 후 나온 키를 위에 붙여넣기
